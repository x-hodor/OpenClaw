## 核心观点

2026年，量子产业已告别早期"概念炒作"，进入**"政策+技术+市场"三重驱动的拐点期**，正在从"实验室走向商业化"。

## 一、政策信号：国家战略级资源倾斜

### 1.1 全球资金投入规模

根据福布斯报道，全球主要经济体在量子技术上的投入已**超过550亿美元**：

| 地区 | 投入详情 |
|------|---------|
| **欧洲** | 英国386亿英镑研发预算中量子获超10亿英镑；西班牙8.08亿欧元(2025-2030)；德国30亿美元；法国20亿美元 |
| **亚洲** | 印度600亿卢比(约7.26亿美元)；新加坡3亿新元(约2.19亿美元) |
| **美国** | 《国家量子计划法案》2019-2023年拨款13亿美元；2025-2029年计划投入27亿美元；能源部6.25亿美元 |
| **中国** | "十五五"规划量子排未来产业第一；央企战新基金510亿首期 |

> **关键信号**：中国"十五五"规划建议中，量子技术位列未来产业**排名第一**，超越核聚变、具身智能等热门赛道。

### 1.2 政府采购：确定性刚需市场

量子通信网络采购已形成**多层级、常态化**格局：

**基础设施建设（建网）**

- 黑龙江哈尔滨：量子城域网建设工程

**核心设备采购（买硬件）**

**垂直行业应用（跑业务）**
- 公安系统："警务通量子密话服务"（终端通话绝对加密）
- 政务外网：电子政务量子安全测评服务
- 科研院所：量子通信技术验证和测试平台

**存量网络运维（持续性收入）**
- 地方数据资源管理局针对"京沪干线"等国家级网络的常态化运维采购
- 涵盖机房运维、光缆维护、7×24小时故障响应

## 二、技术信号：从实验室到工程化

### 2.1 巨头入局验证技术路线

| 巨头 | 布局动作 | 信号意义 |
|------|---------|---------|
| **英伟达** | 投资CQuantinuum、PsiQuantum、QuEra；发布CUDA-Q平台；Grace Blackwell 2100芯片集成 | 量子计算是下一代算力"接棒者"，量超融合降低商业化门槛 |
| **谷歌** | Willow芯片105量子比特；跨越量子纠错阈值；"Quantum Echoes"算法比超算快1.3万倍 | 解决"稳定性"和"可验证性"两大痛点，具备解决实际科学问题潜力 |
| **中国电信** | "天行-287"超导量子计算机；接入"天衍"云平台；全国产化量超融合 | 打通算力底座，验证商业可行性 |
| **华为** | MindSpore Quantum框架和云平台 | 卡位量子计算"操作系统"，软件生态加速成熟 |
| **腾讯/字节** | 腾讯：量子化学与材料仿真；字节：AI for Science新药研发 | 扎根"量子+AI"商业落地 |

### 2.2 工程化突破：三大瓶颈攻克

#### 1) 规模化
- 主流技术路线（超导、离子阱）量子比特突破**百位级**
- 谷歌Willow：105个量子比特
- 已具备化学模拟、药物研发、优化计算等应用探索基础

#### 2) 容错性——跨越"三个9"黄金阈值

**核心概念：两比特门保真度**
- 传统电脑：宏观电压，抗干扰，准确率近100%
- 量子计算机：微观粒子状态脆弱，温度波动或电磁干扰都会导致算错
- **纠错机制门槛**：单次计算准确率必须达到**99.9%**（"三个9"），否则越纠错错误越多

**突破进展**：
- Quantinuum：99.914%
- 东芝双Transmon耦合器：两比特门99.92%，单量子比特门99.98%

> **意义**：当错误率随规模扩大而下降，量子计算结果才具备真正商业实用价值。

#### 3) 工程化

| 核心设备 | 突破进展 |
|---------|---------|
| **稀释制冷机** | 国产打破国际垄断，批量生产10mK级商用制冷机，部分逆向出口海外 |
| **低温线缆** | 从"实验室定制"到"工业级工程化交付" |
| **室温测控系统** | 全国产化，应用于"天衍-287"等整机 |

## 三、市场信号：商业化落地爆发

### 3.1 整机交付：从科研原型到商业订单

| 厂商 | 标志性交付 |
|------|-----------|
| **国盾量子** | 2025年交付合肥"巢湖明月"200比特超导量子计算机及超量融合系统；向海外交付25比特整机（国产超导量子计算技术出海零突破） |
| **本源量子** | "本源悟空"系列向超算中心及高校交付多台，配套自主操作系统 |
| **中科酷原** | 向中国移动交付中国首台100比特中性原子量子计算机"汉源一号"；巴基斯坦出口订单超4000万元 |

### 3.2 商业化趋势

**1) 超量融合成为主流**
- "巢湖明月"、"天衍"平台均采用"经典超算+量子计算"混合部署
- 解决量子计算机无法独立处理复杂通用任务的痛点
- 未来3-5年主要交付形态

**2) 出海破局**
- 国盾量子25比特整机海外交付
- 中科酷原海外订单超4000万元
- 本源量子稀释制冷机国际大额订单
- **信号**：中国量子设备在性价比和技术指标上已具备国际竞争力

**3) 云平台服务与算力订阅**

| 平台 | 运营数据 |
|------|---------|
| **天衍** | 2023年11月对外服务，服务全球50+国家用户，完成60万+实验任务 |
| **本源悟空** | 服务全球139个国家1800万+用户，完成30万+量子计算任务 |

> **商业模式**：从"卖航空母舰"（整机）到"卖船票"（云端按需调用/按年订阅），降低尝试门槛，培育量子生态。

### 3.3 上游核心硬件：国产替代黄金赛道

#### 稀释制冷机

| 指标 | 国产进展 |
|------|---------|
| **市场规模** | 2024年全球约3.54亿美元；2030年预计达百亿美元级别 |
| **技术对标** | 量羲技术Q-400-S、合肥知冷ZL-DR400最低温度7-10mK，与Bluefors 10mK相当 |
| **制冷功率** | 国产100mK温区制冷量450-1000μW，逐步缩小与国际顶尖(1000μW)差距 |
| **成本优势** | 量羲技术均价从2023年602万元/套降至2024年389万元/套，降幅35% |
| **市场占有率** | 量羲技术(被禾信仪器并购)达31% |
| **国际突破** | 本源量子SL400于2025年7月获国际大额订单 |

#### 测控系统

| 指标 | 国产进展 |
|------|---------|
| **市场规模** | 2030年全球预计217.4亿美元(约1522亿人民币) |
| **成本占比** | 量子比特指数级增加后，测控系统将占硬件成本60%-70% |
| **国盾量子** | 2025年6月发布ez-QEngine 2.0，单机箱支持128数据比特+256耦合比特，8台级联完成1000比特操控，价格不到国外一半 |
| **本源量子** | 2025年5月推出"本源天机4.0"，支持500+量子比特，应用于"本源悟空" |
| **国产替代** | 中微达信、耐数电子等加速崛起，形成稳定供应体系 |

### 3.4 量子通信(QKD)：商业模式成熟

**市场规模**：
- 2024年全球约26.7亿美元
- 2030年预计突破75-200亿美元
- 中国市场预计达228.55亿元

**技术演进**：
- **通密一体**：中国电信实现80公里10Tb/s量子信号与经典电信号共纤传输
- **芯片化**：北京大学"未名量子芯网"验证晶圆级集成光量子芯片制造工艺

**应用场景**：
- 国防军事（占比25.65%）
- 国家电网、南方电网核心控制大区部署
- 中国电信与华为合作部署2000公里量子安全OTN专线
- 金融、电力调度、政务、大型赛事（亚冬会）保障

**商业模式转变**：
- 从：政府/国企采购设备
- 到：运营商合作分成模式（国盾量子与电信按用户数支付技术服务费）

## 四、总结与投资跟踪要点

### 4.1 三个关键定调信号

1. **订单量质双升，企业自我造血**——合同负债增长是验证交付情况的先行指标
2. **技术门槛不断跨越**——99.9%保真度阈值突破，工程化能力验证
3. **资本加速与产业整合**——本源量子、国仪量子筹备上市；国际巨头并购加速（IonQ收购Skywater、D-Wave收购Quantum Circuit）

### 4.2 产业链自我强化循环

```
订单 → 工程化迭代 → 成本下降 → 更多订单
```

### 4.3 重点跟踪标的

| 企业 | 核心看点 |
|------|---------|
| **国盾量子** | 整机交付、出海突破、合同负债增长 |
| **量羲技术** | 稀释制冷机国产替代龙头 |
| **本源量子** | 云平台用户量、测控系统、出海订单 |
| **中科酷原** | 中性原子路线、海外订单 |

### 4.4 风险提示

- 技术路线不确定性
- 地缘政治封锁风险
- 商业化进度不及预期
