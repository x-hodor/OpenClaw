# 期货投资基础

> 理解期货本质，认识杠杆与风险

---

## 一、什么是期货？

### 1.1 核心概念

**一句话理解**：期货是**约定未来某个时间以某个价格买卖某物**的标准化合约

```
┌─────────────────────────────────────────────────────────────┐
│                    期货 vs 现货                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  现货交易：                                                  │
│  今天 ────────→ 付钱 ────────→ 拿货                         │
│                                                             │
│  期货交易：                                                  │
│  今天 ────→ 签合约（约定3个月后以100元买1吨铜）             │
│       │                                                     │
│       └──→ 3个月后 ──→ 履约（交钱拿货）或平仓（对冲）       │
│                                                             │
│  期货的两大功能：                                            │
│  1. 套期保值：锁定未来价格，规避波动风险（实体企业用）      │
│  2. 投机：预测价格涨跌，赚取差价（投资者用）                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 期货 vs 股票

```
┌────────────────────────────────────────────────────────────────┐
│  特征          期货                    股票                   │
├────────────────────────────────────────────────────────────────┤
│  交易对象      标准化合约              公司所有权              │
│  持有期限      有到期日（需平仓/交割）  可长期持有             │
│  杠杆          高（通常10-20倍）       无（需配资）           │
│  保证金        只需5-15%资金           需100%资金             │
│  做空          可直接做空              需融券                  │
│  风险          极高（可能爆仓）        较高（最多亏完本金）    │
│  收益          可能极高/极低           中等                    │
│  交易时间      日盘+夜盘               白天                    │
│  适合人群      专业投资者              普通投资者              │
└────────────────────────────────────────────────────────────────┘

⚠️ 警告：期货是高风险投资品种，不适合普通投资者！
```

## 三、期货交易机制

### 3.1 保证金制度

```
┌─────────────────────────────────────────────────────────────┐
│                    保证金制度详解                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  什么是保证金？                                              │
│  └── 交易时只需缴纳合约价值的一部分（5-15%）                │
│                                                             │
│  示例：                                                      │
│  铜期货合约价值 = 70,000元/吨 × 5吨 = 350,000元             │
│  保证金比例 = 10%                                           │
│  实际需缴纳 = 350,000 × 10% = 35,000元                      │
│                                                             │
│  杠杆倍数 = 1 / 10% = 10倍                                  │
│                                                             │
│  盈亏计算：                                                  │
│  铜价涨10% → 盈利 350,000 × 10% = 35,000元                  │
│  相对于本金35,000元 → 收益率100%                            │
│                                                             │
│  ⚠️ 但铜价跌10% → 亏损35,000元 = 本金全部亏完（爆仓）      │
│                                                             │
└─────────────────────────────────────────────────────────────┘

保证金类型：
├── 初始保证金：开仓时缴纳
├── 维持保证金：持仓期间最低要求（低于需补缴）
└── 追加保证金：价格不利变动时需补足
```

### 3.2 每日无负债结算（逐日盯市）

```
┌─────────────────────────────────────────────────────────────┐
│                    每日结算制度                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  每日收盘后结算盈亏：                                        │
│                                                             │
│  例：买入铜期货 @ 70,000元/吨                               │
│                                                             │
│  Day 1: 收盘 71,000 → 盈利 1,000元/吨 × 5吨 = 5,000元      │
│         → 资金账户 +5,000元                                 │
│                                                             │
│  Day 2: 收盘 69,000 → 亏损 2,000元/吨 × 5吨 = 10,000元     │
│         → 资金账户 -10,000元                                │
│                                                             │
│  Day 3: 收盘 68,000 → 亏损 1,000元/吨 × 5吨 = 5,000元      │
│         → 资金账户 -5,000元                                 │
│                                                             │
│  累计：开仓70,000 → 现价68,000                              │
│       亏损 2,000 × 5 = 10,000元                             │
│                                                             │
│  ⚠️ 如果资金不足，会被强制平仓（爆仓）                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.3 强行平仓制度

```
触发强行平仓的情况：

1. 保证金不足
   └── 可用资金 < 0，且未在规定时间内补足

2. 持仓超限
   └── 超过交易所规定的持仓限额

3. 违规操作
   └── 违反交易所规则

4. 到期未平仓
   └── 个人投资者不能参与实物交割
   └── 必须在到期前平仓

强行平仓的后果：
- 损失由投资者承担
- 可能产生穿仓（亏损超过保证金）
- 穿仓需补足差额
```

## 四、期货交易术语

| 术语 | 解释 | 示例 |
|------|------|------|
| **开仓** | 建立新头寸（买入或卖出） | 买入1手铜期货 |
| **平仓** | 了结头寸（反向操作） | 卖出1手铜期货 |
| **多头** | 看涨买入 | 先买后卖 |
| **空头** | 看跌卖出 | 先卖后买 |
| **多头平仓** | 卖出平掉多头 | 卖出之前买的合约 |
| **空头平仓** | 买入平掉空头 | 买入之前卖的合约 |
| **持仓量** | 未平仓合约总数 | 100万手 |
| **成交量** | 当日成交手数 | 50万手 |
| **结算价** | 当日加权平均价 | 用于结算 |
| **交割** | 合约到期实物/现金交割 | 个人不能参与 |
| **基差** | 现货价 - 期货价 | 基差走强/走弱 |
| **升水** | 期货价 > 现货价 | 正常市场 |
| **贴水** | 期货价 < 现货价 | 反向市场 |

## 五、期货投资风险

### 5.1 风险警示

```
┌─────────────────────────────────────────────────────────────┐
│              ⚠️ 期货投资风险警示 ⚠️                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 杠杆风险（最大风险）                                     │
│     ├── 10倍杠杆 → 价格反向波动10% = 本金归零               │
│     ├── 可能产生穿仓（倒欠期货公司钱）                      │
│     └── 案例：2020年原油宝事件，投资者倒欠银行              │
│                                                             │
│  2. 强平风险                                                 │
│     ├── 保证金不足被强制平仓                                │
│     └── 可能在最不利的价位被平仓                            │
│                                                             │
│  3. 流动性风险                                               │
│     ├── 小品种可能无法及时成交                              │
│     └── 极端行情下可能无对手盘                              │
│                                                             │
│  4. 到期风险                                                 │
│     ├── 忘记平仓被强行平仓                                  │
│     └── 个人不能交割，必须提前平仓                          │
│                                                             │
│  5. 操作风险                                                 │
│     ├── 误操作（如方向做反、数量错误）                      │
│     └── 软件故障、网络延迟                                  │
│                                                             │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━   │
│  统计数据：约90%的期货散户投资者亏损                        │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━   │
│                                                             │
└─────────────────────────────────────────────────────────────┘

⚠️ 再次警告：期货不适合普通投资者！
```

## 六、股指期货详解

### 6.1 三大股指期货

```
┌─────────────────────────────────────────────────────────────┐
│                 中国金融期货                                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  品种          标的              合约乘数        保证金      │
├─────────────────────────────────────────────────────────────┤
│  IF（沪深300） 沪深300指数       每点300元      约12%        │
│  IH（上证50）  上证50指数        每点300元      约12%        │
│  IC（中证500） 中证500指数       每点200元      约14%        │
│  IM（中证1000）中证1000指数      每点200元      约15%        │
└─────────────────────────────────────────────────────────────┘

合约价值计算（以IF为例）：
指数点位 4,000点 × 300元/点 = 1,200,000元
保证金：1,200,000 × 12% = 144,000元

盈亏计算：
指数涨1点 → 盈利 1 × 300 = 300元
指数涨100点（2.5%）→ 盈利 30,000元（相对保证金收益率20.8%）
```

### 6.2 股指期货用途

```
对机构投资者的用途：

1. 对冲风险
   持有股票组合，担心下跌
   └── 做空股指期货对冲
   └── 股票跌，期货赚，抵消损失

2. 期现套利
   期货价格 vs 现货价格出现偏差
   └── 买入低估一方，卖出高估一方
   └── 等待价格收敛获利

3. 资产配置
   快速建立/降低股票仓位
   └── 比买卖一篮子股票更高效

对散户的用途：
⚠️ 主要是投机（预测指数涨跌）
⚠️ 风险极高，不建议参与
```

## 七、期货投资的正确姿势

### 7.1 如果你一定要参与

```
必须遵守的原则：

1. 只用闲钱
   └── 亏光不影响生活的资金

2. 严格控制杠杆
   └── 不要超过5倍
   └── 新手建议2-3倍

3. 严格止损
   └── 每笔交易设定止损点
   └── 亏损不超过本金的2-5%

4. 不隔夜（日内交易）
   └── 避免跳空风险
   └── 但交易成本高

5. 轻仓操作
   └── 永远不要满仓
   └── 保留追加保证金的空间

6. 选大品种
   └── 铜、黄金、原油等流动性好的
   └── 避开小品种、临近交割的合约

7. 持续学习
   └── 了解基本面（供需、政策）
   └── 学习技术分析
   └── 模拟盘练习半年以上
```

