## 一、港股市场概况

### 1.1 港股基本特征

```
┌─────────────────────────────────────────────────────────────┐
│                    港股市场特点                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  交易所          香港交易及结算所（HKEX）                    │
│  交易货币        港币（HKD），与美元挂钩                     │
│  交易时间        09:30-12:00，13:00-16:00                   │
│  涨跌停          无涨跌停限制                                │
│  T+0             当日可买卖                                  │
│  结算            T+2（交易后2天结算）                        │
│                                                             │
│  主要指数：                                                  │
│  ├── 恒生指数（HSI）：港股蓝筹代表，50只成份股               │
│  ├── 恒生科技指数：30只科技龙头                              │
│  └── 国企指数（HSCEI）：40只H股                              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 港股 vs A股

```
┌────────────────────────────────────────────────────────────────┐
│  特征          港股                    A股                     │
├────────────────────────────────────────────────────────────────┤
│  投资者结构    机构投资者为主          散户为主                │
│  估值水平      相对较低（PE较低）      相对较高                │
│  分红文化      分红率高（很多4-6%）    分红率较低            │
│  交易制度      T+0，无涨跌停           T+1，有涨跌停         │
│  流动性        小盘股流动性差          相对较好               │
│  做空机制      成熟（可做空个股）      限制较多               │
│  汇率风险      有（港币汇率波动）      无                     │
└────────────────────────────────────────────────────────────────┘

港股特点总结：
✓ 估值洼地：同公司A/H股往往H股更便宜
✓ 国际化：受全球资金影响大
✓ 机构市：更理性，炒作较少
✗ 流动性分化：大盘股流动性好，小盘股差
```

## 二、港股开户方式

### 2.1 两种开户方式对比

```
┌────────────────────────────────────────────────────────────────┐
│  方式          港股通                        直接港股开户      │
├────────────────────────────────────────────────────────────────┤
│  门槛          20个交易日日均资产≥50万       通常无门槛        │
│  开通方式      A股券商开通港股通权限          香港券商开户      │
│  交易标的      恒生大中型股指数成分股          所有港股          │
│               （约500+只，不含仙股）          + 窝轮/牛熊证    │
│  交易货币      人民币自动换汇                 港币             │
│  交易费用      较高（换汇+通道费）            较低             │
│  红利税        20%（H股）/28%（红筹）         10%              │
│  资金门槛      较高                           较低             │
└────────────────────────────────────────────────────────────────┘

选择建议：
- 资金50万+，主要投资大蓝筹 → 港股通（方便、安全）
- 资金较少，或想交易小盘股/衍生品 → 直接港股开户
```

## 三、港股特色投资品种

### 3.1 港股特有产品

```
┌─────────────────────────────────────────────────────────────┐
│                    港股特色产品                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 窝轮（Warrant）                                          │
│     ├── 本质：期权的一种                                     │
│     ├── 特点：杠杆高，到期必须结算                           │
│     ├── 风险：可能归0，时间价值损耗                          │
│     └── 适合：短线投机，风险承受力高                         │
│                                                             │
│  2. 牛熊证                                                   │
│     ├── 本质：带强制收回机制的期权                           │
│     ├── 特点：有收回价，触及即作废                           │
│     ├── 风险：可能瞬间归0（强制收回）                        │
│     └── 适合：短线方向性交易                                 │
│                                                             │
│  3. REITs（房地产信托基金）                                   │
│     ├── 本质：投资房地产的基金                               │
│     ├── 特点：高分红（通常5-8%）                             │
│     ├── 代表：领展房产基金、置富产业信托                     │
│     └── 适合：收息型投资者                                   │
│                                                             │
│  4. 高息股                                                   │
│     ├── 特点：分红率高（5-10%）                              │
│     ├── 代表：汇丰、中移动、长和                             │
│     └── 适合：长期持有收息                                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘

⚠️ 警告：窝轮、牛熊证是极高风险投资品种，不适合新手！
```

## 四、港股投资策略

### 4.1 AH股溢价策略

```
AH股：同一家公司在A股和港股同时上市

┌─────────────────────────────────────────────────────────────┐
│  AH股溢价概念：                                              │
│                                                             │
│  同一家公司：                                                │
│  A股股价：10元人民币                                         │
│  H股股价：7元港币 ≈ 6.5元人民币                              │
│                                                             │
│  AH溢价率 = (A股价格 - H股价格) / H股价格 × 100%            │
│          = (10 - 6.5) / 6.5 = 53.8%                        │
│                                                             │
│  意味着：买H股比买A股便宜53.8%                              │
│                                                             │
└─────────────────────────────────────────────────────────────┘

策略逻辑：
- 买入H股（更便宜）
- 等待AH溢价收窄
- 长期持有收取更高股息

风险：
- 溢价可能长期存在
- 汇率波动影响
- 港股流动性折价
```

### 4.2 港股打新

```
港股IPO特点：
┌─────────────────────────────────────────────────────────────┐
│  特点                   说明                                 │
├─────────────────────────────────────────────────────────────┤
│  中签率高               通常10-100%，热门股也有机会中        │
│  无市值要求             只要有现金即可申购                   │
│  融资打新               可借钱申购（杠杆）                   │
│  暗盘交易               上市前一日下午可交易                 │
│  破发率高               2022-2023年约60%新股破发             │
└─────────────────────────────────────────────────────────────┘

打新策略：
1. 选择优质标的（看保荐人、基石投资者、估值）
2. 现金申购 vs 融资申购（热门股融资，普通股现金）
3. 暗盘观察（暗盘涨得好，首日大概率涨）
4. 及时止损（破发后设定止损线）

注意：港股打新并非稳赚，破发风险大！
```

## 五、港股风险与注意事项

### 5.1 特殊风险

```
┌─────────────────────────────────────────────────────────────┐
│                    港股特有风险                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 汇率风险                                                 │
│     港币与美元挂钩，人民币对港币汇率波动                     │
│     例：人民币升值5%，港股持仓收益减少5%                     │
│                                                             │
│  2. 老千股风险                                               │
│     小市值公司，大股东恶意配股、合股                         │
│     特征：股价低于1港币，市值小，频繁配股                    │
│     防范：远离仙股（股价<1港币）                             │
│                                                             │
│  3. 流动性风险                                               │
│     小盘股可能全天无成交                                     │
│     想卖时卖不掉                                             │
│     防范：专注大盘股                                         │
│                                                             │
│  4. 停牌风险                                                 │
│     港股停牌时间可能很长（数月甚至数年）                     │
│     资金被锁定                                               │
│                                                             │
│  5. 税收风险                                                 │
│     港股通红利税20-28%，高于直接开户的10%                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 5.2 港股通额度限制

```
每日额度限制（足够用，极少用完）：
- 沪股通：520亿人民币
- 深股通：520亿人民币
- 港股通（沪）：420亿人民币
- 港股通（深）：420亿人民币

暂停交易情况：
- 额度用完（极少发生）
- 恶劣天气（台风等）
- 香港或内地节假日不同步
```

## 七、港股市场观察指标

```
重要指标：

1. AH股溢价指数
   - 反映同公司A股比H股贵多少
   - 指数>130表示A股明显溢价
   - 可作为估值参考

2. 港币汇率
   - 联系汇率制：7.75-7.85港币/美元
   - 接近7.85时，可能资金流出香港

3. 港股通资金流向
   - 每日净流入/流出
   - 反映内地资金对港股态度

4. 恒生指数PE/PB
   - 历史分位数判断高低估
   - PE<10通常被认为低估

5. 美国利率
   - 港币跟美元，美联储加息影响港股估值
```
