## 一、K线图的起源与本质

### 1.1 K线的历史

```
┌─────────────────────────────────────────────────────────────┐
│                    K线图的起源                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  起源：日本江户时代（1700年代）                              │
│  创始人：本间宗久（日本米市 trader）                         │
│  最初用途：记录大米价格的每日波动                            │
│                                                             │
│  为什么叫"K线"？                                             │
│  ├── 日本称为"蜡烛线"（Candlestick）                        │
│  ├── 传入西方后称为 Candlestick Chart                       │
│  └── 中文音译为"K线"（取日文"罫線"的发音）                   │
│                                                             │
│  K线为什么有效？                                             │
│  ├── 价格反映一切信息                                        │
│  ├── 历史会重演（人性不变）                                  │
│  └── 趋势一旦形成会持续                                      │
│                                                             │
│  技术分析的三大假设：                                        │
│  1. 市场行为包容消化一切信息                                 │
│  2. 价格以趋势方式演变                                       │
│  3. 历史会重演                                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 K线的构成原理

```
一根K线记录四个价格：

              最高价（High）
                   │
                   ├────── 上影线
                   │
              ┌────┴────┐  ← 实体
              │         │
              │  开盘价  │  ← 如果是阳线，开盘价在下
              │    │    │     如果是阴线，开盘价在上
              │  收盘价  │
              │         │
              └────┬────┘
                   │
                   ├────── 下影线
                   │
              最低价（Low）

阳线（红色/空心）：收盘价 > 开盘价
├── 表示当日上涨
├── 实体越大，上涨力量越强
└── 说明买方力量强于卖方

阴线（绿色/实心）：收盘价 < 开盘价  
├── 表示当日下跌
├── 实体越大，下跌力量越强
└── 说明卖方力量强于买方

为什么用实体表示开盘收盘？
└── 这两个价格最重要，代表市场共识
```

### 1.3 影线背后的市场心理

```
┌─────────────────────────────────────────────────────────────┐
│                    影线的市场含义                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  上影线（从实体到最高价）                                    │
│  ├── 形成过程：价格冲高后被压回                              │
│  ├── 含义：上方有阻力，卖方在高位阻击                        │
│  └── 上影线越长，阻力越大                                    │
│                                                             │
│  下影线（从实体到最低价）                                    │
│  ├── 形成过程：价格探底后被拉回                              │
│  ├── 含义：下方有支撑，买方在低位承接                        │
│  └── 下影线越长，支撑越强                                    │
│                                                             │
│  实例分析：                                                  │
│                                                             │
│  长上影线阳线：                                              │
│        ┬                                                     │
│        │  ← 上影线很长                                       │
│       ███  ← 但收盘仍涨                                      │
│       ███                                                    │
│        │                                                     │
│        ┴                                                     │
│  解读：冲高遇阻，但买方仍占上风，需警惕次日回调              │
│                                                             │
│  长下影线阳线：                                              │
│        ┬                                                     │
│        │                                                     │
│       ███                                                    │
│       ███  ← 实体较大                                        │
│        │  ← 下影线很长                                       │
│        ┴                                                     │
│  解读：先跌后涨，买方在低位强力承接，看涨信号                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 实体大小的意义

```
实体大小的市场含义：

实体很大（>3%）：
├── 当天力量对比悬殊
├── 趋势方向明确
└── 后续可能延续

实体中等（1-3%）：
├── 正常波动
├── 力量对比均衡
└── 需结合其他信号

实体很小（<1%）：
├── 多空僵持
├── 力量均衡
├── 可能是变盘前兆
└── 特别是配合长影线时（十字星）

实体变化趋势：
上升趋势中，阳线实体逐渐变小 = 上涨动力减弱，警惕
下跌趋势中，阴线实体逐渐变小 = 下跌动力减弱，可能反弹
```

## 四、趋势与支撑阻力

### 4.1 趋势理论

```
┌─────────────────────────────────────────────────────────────┐
│                    趋势的定义与类型                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  上升趋势：高点越来越高，低点也越来越高                      │
│                                                             │
│        ┌──┐                                                │
│       ┌┘  └┐                                               │
│      ┌┘    └┐    每一个高点 > 前一个高点                   │
│     ┌┘      └┐   每一个低点 > 前一个低点                   │
│    ┌┘        └┐                                            │
│  ──┘          └────→                                       │
│                                                             │
│  下降趋势：低点越来越低，高点也越来越低                      │
│                                                             │
│    ┌┐                                                      │
│   ┌┘└┐                                                     │
│  ┌┘  └┐      每一个低点 < 前一个低点                       │
│  ┘    └┐     每一个高点 < 前一个高点                       │
│         └┐                                                 │
│          └────→                                            │
│                                                             │
│  横盘整理：高点和低点在一个区间内波动                        │
│                                                             │
│    ┌───┐                                                   │
│   ┌┘   └┐                                                  │
│   │     │    价格在上下轨之间震荡                          │
│  ┌┘     └┐                                                 │
│  └───┬───┘                                                 │
│                                                             │
│  趋势的力量：                                                │
│  ├── 趋势一旦形成，会持续一段时间                            │
│  ├── 不要逆势操作（不要接下跌的飞刀）                        │
│  └── 顺势而为是技术分析的核心                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 支撑与阻力

```
┌─────────────────────────────────────────────────────────────┐
│                    支撑与阻力的原理                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  支撑（Support）：价格下跌时遇到的买盘                       │
│  阻力（Resistance）：价格上涨时遇到的卖盘                    │
│                                                             │
│  为什么形成支撑/阻力？                                       │
│                                                             │
│  1. 心理关口                                                 │

│     └── 历史高低点                                           │
│                                                             │
│  2. 成本集中区                                               │
│     ├── 前一波上涨/下跌的起点                                │
│     └── 密集成交区                                           │
│                                                             │

│     ├── 移动平均线成为动态支撑/阻力                          │

│  支撑位一旦被跌破，就变成阻力位                              │
│                                                             │
│  图示：                                                      │
│                                                             │
│        阻力位                                                │
│       ─────────  ← 多次冲不过                                │
│          │                                                   │
│          │ 突破                                              │
│          ▼                                                   │
│       ─────────  ← 变成支撑位                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 五、成交量分析

### 5.1 价量关系原理

```
┌─────────────────────────────────────────────────────────────┐
│                    成交量（Volume）的意义                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  成交量 = 市场参与度 = 资金流入流出                          │
│                                                             │
│  量价配合原则：                                              │
│                                                             │
│  1. 价涨量增（健康上涨）                                     │
│     ├── 价格上涨，成交量放大                                 │

│                                                             │
│  2. 价涨量缩（上涨乏力）                                     │
│     ├── 价格上涨，成交量萎缩                                 │
│     ├── 说明买盘不足，可能是诱多                             │
│     └── 警惕回调                                             │
│                                                             │
│  3. 价跌量增（恐慌抛售）                                     │
│     ├── 价格下跌，成交量放大                                 │
│     ├── 说明卖盘涌出，恐慌情绪                               │
│     └── 可能是底部信号（也可能是下跌中继）                   │
│                                                             │
│  4. 价跌量缩（无人接盘）                                     │
│     ├── 价格下跌，成交量萎缩                                 │
│     ├── 说明观望情绪浓厚                                     │
│     └── 下跌趋势可能持续                                     │
│                                                             │
│  天量与地量：                                                │
│  ├── 天量：成交量突然放大到历史高位                          │
│  │        可能是见顶信号（主力出货）                         │
│  └── 地量：成交量萎缩到历史低位                              │
│           可能是见底信号（地量见地价）                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 六、技术指标原理

### 6.1 移动平均线（MA）

```
移动平均线的原理：

MA5 = 过去5天收盘价的平均值
MA10 = 过去10天收盘价的平均值
...

作用：
1. 平滑价格，看清趋势
2. 提供动态支撑/阻力
3. 金叉死叉产生交易信号

常用均线：
├── 短期：MA5、MA10（5日线、10日线）
├── 中期：MA20、MA60（月线、季线）
└── 长期：MA120、MA250（半年线、年线）

金叉与死叉：

金叉（买入信号）：
短期MA上穿长期MA
例：MA5上穿MA10
含义：短期趋势转强

死叉（卖出信号）：
短期MA下穿长期MA
例：MA5下穿MA10
含义：短期趋势转弱

注意：
- 震荡市均线会频繁金叉死叉，产生假信号
- 结合其他指标使用
```

### 6.2 MACD原理

```
MACD（Moving Average Convergence Divergence）

计算公式：
1. DIF（快线）= EMA12 - EMA26
2. DEA（慢线）= DIF的9日EMA
3. MACD柱状图 = (DIF - DEA) × 2

原理：
- 两条不同周期的EMA的差距
- 差距扩大 = 趋势加速
- 差距缩小 = 趋势减速

信号：
- 金叉：DIF上穿DEA，买入信号
- 死叉：DIF下穿DEA，卖出信号
- 背离：价格创新高，MACD不创新高 = 顶背离（看跌）

使用注意：
- 滞后性（基于均线）
- 震荡市效果差
- 大趋势指标
```

## 七、技术分析的局限性

```
┌─────────────────────────────────────────────────────────────┐
│                    技术分析的局限                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 滞后性                                                   │
│     ├── K线、均线都是基于历史价格                            │
│     ├── 信号出现时，价格已经变动                             │
│     └── 不能预测突发事件                                     │
│                                                             │
│  2. 假信号                                                   │
│     ├── 震荡市中，指标频繁金叉死叉                           │
│     ├── 形态可能失败                                         │
│     └── 需要结合其他方法验证                                 │
│                                                             │
│  3. 自我实现与反身性                                         │
│     ├── 太多人用同样的方法，可能失效                         │
│     ├── 主力可能故意做出技术图形骗线                         │
│     └── 没有永远有效的方法                                   │
│                                                             │
│  4. 无法判断价值                                             │
│     ├── 技术分析只关心价格，不关心公司                       │
│     ├── 无法区分好公司和坏公司                               │
│     └── 需要结合基本面分析                                   │
│                                                             │
│  正确态度：                                                  │
│  ├── 技术分析是工具，不是圣杯                                │
│  ├── 结合基本面、资金面、消息面                              │
│  ├── 做好风险管理，设好止损                                  │
│  └── 长期看，价值决定价格                                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
