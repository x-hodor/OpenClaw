# 投资分析方法

> 基本面分析与技术分析入门

---

## 一、投资分析两大流派

```
┌─────────────────────────────────────────────────────────────┐
│                    投资分析方法                              │
└─────────────────────────────────────────────────────────────┘

流派一：基本面分析（Fundamental Analysis）
├── 核心逻辑：股票价格由公司内在价值决定
├── 分析方法：研究公司财务、行业、宏观经济
├── 投资周期：中长期（1年以上）
├── 代表人物：巴菲特、彼得·林奇
└── 适合人群：有耐心、愿意研究的投资者

流派二：技术分析（Technical Analysis）
├── 核心逻辑：价格反映一切信息，历史会重演
├── 分析方法：研究价格走势、成交量、图表形态
├── 投资周期：短期（几天到几周）
├── 代表人物：杰西·利弗莫尔
└── 适合人群：时间充裕、反应快的投资者

注意：
- 没有哪一种方法绝对更好
- 很多成功投资者两者结合使用
- 关键是找到适合自己的方法并坚持
```

## 二、基本面分析

### 2.1 财务分析核心指标

```
┌─────────────────────────────────────────────────────────────┐
│                  财务分析关键指标                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  盈利能力指标                                                │
│  ├── ROE（净资产收益率）= 净利润 / 净资产                    │
│  │   └── 优秀标准：>15%，连续多年                          │
│  │   └── 巴菲特最看重的指标                                │
│  │                                                         │
│  ├── 毛利率 = (营收 - 成本) / 营收                          │
│  │   └── 反映产品竞争力                                    │
│  │   └── 茅台90%+，制造业20-40%                            │
│  │                                                         │
│  └── 净利率 = 净利润 / 营收                                 │
│      └── 反映最终盈利能力                                  │
│                                                             │
│  估值指标                                                    │
│  ├── PE（市盈率）= 股价 / 每股收益                          │
│  │   └── 投资回本年限                                      │
│  │   └── 沪深300平均PE约12-15倍                            │
│  │                                                         │
│  ├── PB（市净率）= 股价 / 每股净资产                        │
│  │   └── <1可能被低估，>5可能高估                          │
│  │   └── 银行股通常PB<1                                    │
│  │                                                         │
│  └── PEG = PE / 盈利增长率                                  │
│      └── <1表示估值合理，>2可能高估                        │
│      └── 彼得·林奇常用指标                                 │
│                                                             │
│  偿债能力指标                                                │
│  ├── 资产负债率 = 负债 / 资产                               │
│  │   └── <60%较安全，>80%风险高                            │
│  │                                                         │
│  └── 流动比率 = 流动资产 / 流动负债                         │
│      └── >2表示短期偿债能力强                              │
│                                                             │
│  成长能力指标                                                │
│  ├── 营收增长率（同比）                                     │
│  ├── 净利润增长率（同比）                                   │
│  └── 连续多年>20%为高增长公司                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘

财务分析要点：
1. 看3-5年数据，不是单一年份
2. 与同行业对比
3. 关注趋势变化
```

### 2.2 行业分析框架

```
波特五力模型：

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                    现有竞争者                                │
│                    （竞争激烈程度）                          │
│                         │                                   │
│     潜在进入者 ─────────┼───────── 替代品威胁               │
│    （进入门槛）          │          （被替代风险）           │
│                         │                                   │
│                   供应商议价能力                             │
│                   购买者议价能力                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘

分析要点：
- 竞争格局：垄断/寡头/充分竞争
- 行业生命周期：导入期/成长期/成熟期/衰退期
- 政策影响：支持/限制/中性
- 技术变革：颠覆风险

行业分类：
├── 周期性行业（业绩随经济波动）
│   └── 钢铁、煤炭、有色金属、房地产
├── 防御性行业（业绩相对稳定）
│   └── 公用事业、食品、医药
├── 成长性行业（高增长）
│   └── 科技、新能源、生物医药
└── 金融类（特殊监管）
    └── 银行、保险、证券
```

### 2.3 宏观分析要点

```
影响股市的宏观因素：

1. 经济增长
   ├── GDP增速
   └── 企业盈利与GDP正相关

2. 利率政策
   ├── 降息 → 利好股市（资金成本低）
   └── 加息 → 利空股市（估值压缩）

3. 通胀水平
   ├── 温和通胀（2-3%）→ 利好
   └── 高通胀 → 央行加息，利空

4. 汇率变动
   ├── 本币升值 → 利好（外资流入）
   └── 本币贬值 → 利空（资本外流）

5. 政策环境
   ├── 货币政策：宽松/紧缩
   ├── 财政政策：扩张/收缩
   └── 产业政策：扶持/限制

6. 国际环境
   ├── 地缘政治
   ├── 国际贸易
   └── 全球资金流向
```

## 三、技术分析

### 3.1 趋势理论

```
核心观点：价格沿趋势运动

三种趋势：
┌─────────────────────────────────────────────────────────────┐
│  趋势类型      时间周期         特点                        │
├─────────────────────────────────────────────────────────────┤
│  主要趋势      数月到数年       长期方向，最重要            │
│  （大趋势）     牛市/熊市                                   │
├─────────────────────────────────────────────────────────────┤
│  次要趋势      数周到数月       大趋势中的回调/反弹         │
│  （中期调整）    幅度通常为1/3-2/3                         │
├─────────────────────────────────────────────────────────────┤
│  短期趋势      数天到数周       噪音多，最难把握            │
│  （日常波动）    可忽略                                    │
└─────────────────────────────────────────────────────────────┘

趋势判断：
├── 上升趋势：高点越来越高，低点越来越高
├── 下降趋势：低点越来越低，高点越来越低
└── 横盘整理：高点低点在一个区间内
```

### 3.2 常用技术指标

```
移动平均线（MA）：
├── MA5/MA10：短期趋势
├── MA20/MA60：中期趋势
└── MA120/MA250：长期趋势

金叉/死叉：
├── 金叉：短期MA上穿长期MA → 买入信号
└── 死叉：短期MA下穿长期MA → 卖出信号

MACD（指数平滑异同平均线）：
├── DIF线（快线）
├── DEA线（慢线）
├── MACD柱状图
└── 金叉+零轴上方 = 强势上涨

RSI（相对强弱指标）：
├── 0-100之间波动
├── >70：超买，可能回调
├── <30：超卖，可能反弹
└── 50：多空分界线

KDJ（随机指标）：
├── K线、D线、J线
├── 金叉买入，死叉卖出
└── 同样看超买超卖

成交量（VOL）：
├── 价涨量增：上涨趋势确认
├── 价涨量缩：上涨乏力
├── 价跌量增：恐慌抛售
└── 价跌量缩：整理或见底
```

### 3.3 K线形态

```
反转形态（顶部）：
├── 双顶（M头）
├── 头肩顶
├── 圆弧顶
└── 三只乌鸦

反转形态（底部）：
├── 双底（W底）
├── 头肩底
├── 圆弧底
└── 早晨之星

持续形态：
├── 三角形（收敛）
├── 旗形
├── 楔形
└── 矩形（整理）

重要提醒：
- 形态需要结合成交量确认
- 形态失败很常见，必须设止损
- 大周期形态比小周期更可靠
```

## 四、分析方法选择建议

```
┌─────────────────────────────────────────────────────────────┐
│                    分析方法选择                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  如果你是价值投资者（如巴菲特）：                            │
│  └── 主要看基本面：ROE、现金流、护城河                      │
│  └── 少看或不看技术分析                                    │
│  └── 持有周期：3-5年以上                                   │
│                                                             │
│  如果你是成长投资者（如彼得·林奇）：                        │
│  └── 主要看：增长率、行业空间、管理层                       │
│  └── 结合PEG估值                                           │
│  └── 持有周期：1-3年                                       │
│                                                             │
│  如果你是趋势投资者：                                       │
│  └── 主要看：趋势方向、均线系统                            │
│  └── 设好止损，顺势而为                                    │
│  └── 持有周期：数周到数月                                  │
│                                                             │
│  如果你是短线交易者：                                       │
│  └── 主要看：技术形态、成交量、资金流向                    │
│  └── 严格止损，快进快出                                    │
│  └── 持有周期：几天到几周                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘

建议：
- 新手从基本面分析入手
- 技术分析作为辅助工具
- 不要盲目迷信任何单一方法
```
