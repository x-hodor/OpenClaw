## 二、美股开户方式

### 2.1 开户渠道

```
┌────────────────────────────────────────────────────────────────┐
│  渠道              特点                  适合人群              │
├────────────────────────────────────────────────────────────────┤
│                                                             │
│  直接美股券商                                      │

- 身份证

## 三、美股投资方式

### 3.1 直接投资股票

```
适合：有一定研究能力，追求超额收益

优势：
✓ 可精选个股
✓ 收益上限高
✓ 参与公司治理（股东大会投票）

风险：
✗ 个股风险集中
✗ 需要研究能力
✗ 时间成本高

推荐标的类型：
├── 大盘蓝筹：苹果、微软、谷歌（稳健）
├── 科技成长：英伟达、特斯拉、Meta（高成长）
├── 中概股：拼多多、阿里、京东（熟悉业务）
└── ETF：SPY、QQQ（分散投资）
```

✓ 分散投资，降低个股风险

✓ 透明度高，持仓每日公布
✓ 流动性好，像股票一样交易
```

## 五、美股投资要点

### 5.1 财报季关注

```
美股财报季（每季度）：
- 1月：Q4财报（重点，全年总结）
- 4月：Q1财报
- 7月：Q2财报
- 10月：Q3财报

财报关键指标：
┌─────────────────────────────────────────────────────────────┐
│  指标              含义                                      │
├─────────────────────────────────────────────────────────────┤
│  EPS（每股收益）    最重要指标，同比/环比变化                │
│  Revenue（营收）    收入规模，增长率                        │
│  Guidance（指引）   未来季度/全年预期                       │
│  MAU/DAU           用户数据（互联网公司）                   │
│  ARPU              每用户平均收入                           │
│  Free Cash Flow    自由现金流                               │
└─────────────────────────────────────────────────────────────┘

财报后股价波动大，注意风险控制
```

### 5.2 美联储政策影响

```
美联储（Fed）政策是美股最重要的宏观因素：

利率政策：
┌─────────────────────────────────────────────────────────────┐
│  环境          政策          对美股影响                      │
├─────────────────────────────────────────────────────────────┤
│  加息周期      提高利率      股市承压，成长股下跌更多        │
│  降息周期      降低利率      股市上涨，估值扩张              │
│  量化宽松      购买债券      流动性充裕，股市上涨            │
│  缩表          缩减负债表    流动性收紧，股市承压            │
└─────────────────────────────────────────────────────────────┘

关注指标：
- 联邦基金利率
- CPI/PCE（通胀数据）
- 非农就业数据
- 鲍威尔讲话（美联储主席）
```

## 六、美股风险与注意事项

### 6.1 主要风险

```
┌─────────────────────────────────────────────────────────────┐
│                    美股投资风险                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  汇率风险                                                    │
│  ├── 人民币对美元贬值，投资收益减少                          │
│  ├── 例：股票涨10%，汇率跌5%，实际收益仅5%                   │
│  └── 对冲：使用外汇远期，或接受长期持有                      │
│                                                             │
│  政策风险                                                    │
│  ├── 中美贸易战影响中概股                                    │
│  ├── 美国对华制裁影响特定行业                                │
│  └── 国内政策影响（如教培、数据安全）                        │
│                                                             │
│  税收风险                                                    │
│  ├── 股息税30%（W-8BEN可减至10%）                           │
│  ├── 资本利得税（非美居民通常无）                            │
│  └── 遗产税（外国人买美股免）                                │
│                                                             │
│  时差风险                                                    │
│  ├── 美股夜间交易，影响睡眠                                  │
│  ├── 重大事件发生时无法及时反应                              │
│  └── 建议：设置条件单，避免盯盘                              │
│                                                             │
│  信息不对称                                                  │
│  ├── 英文信息为主                                            │
│  ├── 国内投资者获取信息滞后                                  │
│  └── 建议：关注英文财报和新闻                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 6.2 投资者适当性

```
美股投资适合：
✓ 有境外资产配置需求
✓ 能承受汇率波动
✓ 有一定英文阅读能力
✓ 了解美股交易规则
✓ 有长期投资视野

不适合：
✗ 无法承受本金损失
✗ 需要频繁使用资金
✗ 无法接受夜间交易时间
✗ 对汇率风险极度敏感
```
