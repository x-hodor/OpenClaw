## 二、仓位管理

### 2.1 仓位控制原则

```
┌─────────────────────────────────────────────────────────────┐
│                    仓位管理原则                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  永远不要满仓                                               │
│  └── 保留现金应对突发情况                                   │
│  └── 建议最高仓位：80%                                      │
│                                                             │
│  分仓策略                                                   │
│  ├── 不要把所有钱买一只股票                                 │
│  ├── 单只股票不超过总资金的20-30%                           │
│  └── 建议持有5-10只不同行业的股票                           │
│                                                             │
│  金字塔建仓                                                 │
│  ├── 首次建仓：计划仓位的30%                                │
│  ├── 下跌加仓：再跌5-10%，加仓40%                           │
│  └── 继续下跌：加仓剩余的30%                                │
│  └── 避免：一次性满仓                                       │
│                                                             │
│  动态再平衡                                                 │
│  ├── 定期检查各资产比例                                     │
│  ├── 某资产涨太多 → 卖出部分，买入其他                      │
│  └── 保持目标配置比例                                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 凯利公式（理论参考）

```
凯利公式：f = (bp - q) / b

f: 最优仓位比例
b: 赔率（盈利/亏损）
p: 胜率
q: 败率（1-p）

例子：
胜率60%，赔率为2（赚2亏1）
f = (2×0.6 - 0.4) / 2 = 0.4 = 40%

实际应用建议：
- 凯利公式给出的是理论最优值
- 实际投资建议用"半凯利"或"1/4凯利"
- 上述例子实际操作20%仓位更安全

警告：
- 不要高估自己的胜率
- 市场条件会变化
- 保守永远是正确的
```

## 三、止损策略

### 3.1 止损的重要性

```
┌─────────────────────────────────────────────────────────────┐
│                    为什么要止损                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  没有止损的后果：                                           │
│                                                             │
│  股票A：100元买入 → 跌到80元（-20%）                        │
│         "跌这么多了，应该会反弹"                            │
│         → 跌到50元（-50%）                                  │
│         "我就不信还能跌"                                    │
│         → 跌到20元（-80%）                                  │
│         最终割肉或套牢多年                                  │
│                                                             │
│  股票B：100元买入，设止损-10%                               │
│         跌到90元自动卖出                                    │
│         亏损10%，但保住了大部分本金                         │
│         资金可用于其他机会                                  │
│                                                             │
│  结论：                                                      │
│  - 止损是承认错误的勇气                                     │
│  - 小亏好过深套                                             │
│  - 止损让资金保持流动性                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 止损方法

```
常用止损方法：

1. 固定比例止损
   └── 亏损达到X%就卖出
   └── 常见设置：-7%、-10%、-15%
   └── 优点：简单执行
   └── 缺点：可能因正常波动被洗出

2. 技术位止损
   └── 跌破重要支撑位止损
   └── 如：跌破均线、前期低点、趋势线
   └── 优点：符合技术分析
   └── 缺点：主观性强

3. 时间止损
   └── 买入后一段时间不涨就卖出
   └── 如：持有2周不涨，说明判断错误
   └── 优点：避免机会成本
   └── 缺点：可能错过慢牛股

4. 波动率止损（ATR止损）
   └── 根据股票波动特性设置
   └── 波动大的股票给更大空间
   └── 更科学合理

建议：
- 短线：-5%到-7%
- 中线：-10%到-15%
- 长线：-20%到-30%
- 一旦触发，坚决执行
```

### 3.3 止盈策略

```
止盈比止损更难：
- 贪婪：想赚更多
- 恐惧：担心回撤

常用止盈方法：

1. 目标价止盈
   └── 达到预期收益就卖出
   └── 例：设定+20%目标，达到即卖出

2. 移动止盈（跟踪止损）
   └── 盈利后逐步提高止损线
   └── 例：涨20%后，止损设为+10%
   └── 既保护利润，又让利润奔跑

3. 分批止盈
   └── 涨20%卖出30%
   └── 涨30%卖出30%
   └── 涨50%卖出剩余40%
   └── 锁定部分利润，留部分博更大

4. 技术信号止盈
   └── 出现顶部形态
   └── 跌破上升趋势线
   └── 超买指标出现

心理建议：
- 接受"卖飞"是正常的
- 没有谁能卖在最高点
- 落袋为安比坐过山车好
```

## 四、资产配置

### 4.1 分散投资

```
不要把鸡蛋放在一个篮子里

分散维度：
┌─────────────────────────────────────────────────────────────┐
│  维度              分散方式                                  │
├─────────────────────────────────────────────────────────────┤
│  资产类别          股票 + 债券 + 现金 + 商品                │
│  行业              科技 + 消费 + 医疗 + 金融 + ...          │
│  地区              A股 + 港股 + 美股 + 其他市场             │
│  市值              大盘股 + 中盘股 + 小盘股                 │
│  风格              价值股 + 成长股                          │
│  时间              分批买入，不要一次性投入                 │
└─────────────────────────────────────────────────────────────┘

注意：
- 分散过度也不好（持有太多股票跟踪不过来）
- 建议：5-10只股票，3-5种资产类别
- 相关性低的资产才能真正分散风险
```

### 4.2 股债配置策略

```
按年龄配置：

股票比例 ≈ 100 - 年龄

25岁：75%股票 + 25%债券
40岁：60%股票 + 40%债券
60岁：40%股票 + 60%债券

根据风险承受能力调整：
┌─────────────────────────────────────────────────────────────┐
│  风险偏好      股票      债券      适合人群                  │
├─────────────────────────────────────────────────────────────┤
│  保守型        20%       80%       退休人群、极度厌恶风险    │
│  稳健型        40%       60%       中年、有家庭负担          │
│  平衡型        60%       40%       有一定经验的投资者        │
│  进取型        80%       20%       年轻、能承受波动          │
│  激进型        100%      0%        专业投资者                │
└─────────────────────────────────────────────────────────────┘

再平衡：
- 每年检查一次配置比例
- 偏离目标5%以上时调整
- 卖出涨得多的，买入跌得多的
- 低买高卖的纪律执行
```

## 五、心理管理

### 5.1 常见投资心理误区

```
┌─────────────────────────────────────────────────────────────┐
│                   投资心理陷阱                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  贪婪                                                        │
│  ├── 涨了还想涨，不卖                                       │
│  ├── 追高买入，被套在山顶                                   │
│  └── 对策：设定目标价，严格执行                             │
│                                                             │
│  恐惧                                                        │
│  ├── 跌了一点就割肉                                         │
│  ├── 低位不敢加仓                                           │
│  └── 对策：相信价值，逆向投资                               │
│                                                             │
│  从众（羊群效应）                                            │
│  ├── 大家都买我也买                                         │
│  ├── 恐慌时跟着卖                                           │
│  └── 对策：独立思考，人弃我取                               │
│                                                             │
│  过度自信                                                    │
│  ├── 赚了几次就觉得自己是股神                               │
│  ├── 加大仓位，满仓梭哈                                     │
│  └── 对策：永远敬畏市场                                     │
│                                                             │
│  锚定效应                                                    │
│  ├── "我10块买的，跌到8块就不卖，等回10块"                 │
│  ├── 成本价成了心理锚点                                     │
│  └── 对策：关注价值，不是成本                               │
│                                                             │
│  损失厌恶                                                    │
│  ├── 亏钱的股票死拿（不愿面对损失）                         │
│  ├── 赚钱的股票很快卖掉（锁定利润）                         │
│  └── 结果：亏大赚小                                         │
│  └── 对策：反过来，截断亏损，让利润奔跑                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 5.2 投资纪律

```
建立并遵守投资纪律：

买入纪律：
□ 只买自己了解的公司
□ 有安全边际才买入
□ 分批建仓，不一次性满仓
□ 设定止损位

持有纪律：
□ 定期检查基本面是否变化
□ 不被短期波动影响
□ 不频繁交易

卖出纪律：
□ 达到止损位坚决卖出
□ 基本面恶化卖出
□ 估值过高卖出
□ 有更好的投资机会时换仓

情绪管理：
□ 不因为涨跌而兴奋或沮丧
□ 每天看盘不超过1小时
□ 重大决策隔夜再确认
□ 保持独立思考
```

## 六、风险管理 checklist

```
投资前检查：
□ 这笔钱是3年以上不用的闲钱
□ 已预留6个月生活费
□ 已配置保险（医疗、意外）
□ 明确自己的风险承受能力
□ 设定了投资目标和期限

投资中检查：
□ 单只股票仓位<30%
□ 总仓位<80%
□ 每只股票设了止损
□ 分散在不同行业
□ 定期复盘投资逻辑

市场极端情况预案：
□ 如果大盘跌30%怎么办？
□ 如果个股暴雷怎么办？
□ 如果急需用钱怎么办？

紧急应对：
□ 设置了自动止损
□ 了解如何快速平仓
□ 有备用资金渠道
```
