## 一、什么是股票？

### 1.1 核心概念

**一句话理解**：股票是公司所有权的凭证，持有股票 = 成为公司股东

```
┌─────────────────────────────────────────────────────────────┐
│                     股票的本质                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   公司需要资金发展 ──→ 发行股票 ──→ 投资者购买               │
│         │                              │                    │
│         │                              │                    │
│         ▼                              ▼                    │
│   获得资金用于经营               获得所有权+收益权           │
│                                                             │
│   股东权利：                                                 │
│   1. 分红权（公司盈利后分得利润）                            │
│   2. 投票权（参与公司重大决策）                              │
│   3. 转让权（在二级市场买卖股票）                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 股票的收益来源

```
股票收益 = 资本利得 + 股息收入

┌─────────────────────────────────────────────────────────────┐
│  收益类型          说明                    示例              │
├─────────────────────────────────────────────────────────────┤
│  资本利得          低买高卖的差价          10元买入，         │
│  (价差收益)        主要收益来源            15元卖出 = 赚5元   │
├─────────────────────────────────────────────────────────────┤
│  股息收入          公司分红               每股分红0.5元，      │
│  (现金分红)        持有1000股 = 500元     持股10万股 = 5万   │
└─────────────────────────────────────────────────────────────┘

收益计算示例：
买入：1000股 × 10元 = 10,000元
卖出：1000股 × 15元 = 15,000元
分红：1000股 × 0.5元 = 500元
────────────────────────────────
总收益：5,000 + 500 = 5,500元 (收益率 55%)
```

## 二、股票市场结构

### 2.1 市场层次

```
┌─────────────────────────────────────────────────────────────┐
│                    股票市场层次                              │
└─────────────────────────────────────────────────────────────┘

一级市场（发行市场）
├── IPO（首次公开发行）
│   └── 公司第一次向公众发售股票
├── 增发
│   └── 已上市公司再次发行股票
└── 配股
    └── 向现有股东按比例发售新股

二级市场（交易市场）
├── 主板市场
│   ├── 大型成熟企业
│   └── 准入门槛高
├── 创业板/科创板
│   ├── 成长型企业
│   └── 创新科技企业
├── 新三板
│   └── 中小微企业
└── 场外市场（OTC）
    └── 非上市股票交易
```

```
┌─────────────────────────────────────────────────────────────────────────┐
```

## 三、股票基础术语

### 3.1 核心术语表

| 术语 | 解释 | 示例 |
|------|------|------|
| **股价** | 股票当前交易价格 | 100元/股 |
| **市值** | 股价 × 总股本 | 股价100元 × 1亿股 = 100亿市值 |
| **市盈率(PE)** | 股价 ÷ 每股收益 | PE=20表示投资20年回本 |
| **市净率(PB)** | 股价 ÷ 每股净资产 | PB<1可能被低估 |
| **成交量** | 某时段交易股数 | 100万股 |
| **成交额** | 某时段交易金额 | 1亿元 |
| **换手率** | 成交量 ÷ 流通股本 | 5%表示流通筹码换了5% |
| **涨停/跌停** | 当日最高/最低涨幅限制 | A股±10%，ST股±5% |
| **多头** | 看涨买入方 | 认为股价会涨 |
| **空头** | 看跌卖出方 | 认为股价会跌 |
| **牛市** | 市场整体上涨 | 上证指数持续上涨 |
| **熊市** | 市场整体下跌 | 上证指数持续下跌 |

## 四、股票投资基础流程

```
┌─────────────────────────────────────────────────────────────┐
│                   股票投资基础流程                           │
└─────────────────────────────────────────────────────────────┘

Step 1: 开户
    │
    ├── 选择券商（佣金费率、交易软件、服务）
    ├── 准备材料（身份证、银行卡）
    └── 完成开户（线上/线下）
    │
    ▼
Step 2: 入金
    │
    ├── 绑定银行卡
    └── 转入投资资金
    │
    ▼
Step 3: 选股
    │
    ├── 基本面分析（公司财务、行业前景）
    ├── 技术面分析（K线、指标）
    └── 确定投资标的
    │
    ▼
Step 4: 交易
    │
    ├── 下单（限价单/市价单）
    ├── 成交确认
    └── 持仓管理
    │
    ▼
Step 5: 监控与调整
    │
    ├── 跟踪持仓表现
    ├── 止损/止盈
    └── 再平衡
```

## 五、投资风险提示

### 5.1 主要风险类型

```
┌─────────────────────────────────────────────────────────────┐
│                     股票投资风险                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  市场风险（系统性风险）                                      │
│  ├── 整个市场环境变差，所有股票都跌                          │
│  └── 无法通过分散投资消除                                   │
│                                                             │
│  个股风险（非系统性风险）                                    │
│  ├── 公司经营不善、财务造假                                 │
│  └── 可以通过分散投资降低                                   │
│                                                             │
│  流动性风险                                                  │
│  ├── 想卖时卖不掉（小盘股、停牌）                           │
│                                                             │
│  政策风险                                                    │
│  ├── 行业监管政策变化（如教培、房地产）                     │
│                                                             │
│  汇率风险（港股、美股）                                      │
│  ├── 人民币对美元/港币贬值，投资收益被侵蚀                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 5.2 新手常见误区

| 误区 | 正确认知 |
|------|----------|
| "股票能快速致富" | 长期平均年化收益 8-10%，暴富是小概率 |
| "买涨得多的股票" | 追涨杀跌是高亏损的主要原因 |
| "听消息炒股" | 消息到你这里时，价格已反映 |
| "跌了就死扛" | 止损是风险管理的重要一环 |
| "All in 一只股票" | 分散投资，不要把鸡蛋放一个篮子 |
| "频繁交易" | 交易成本高，且容易追高杀跌 |
